# ItHub-Menu

Современное веб-приложение для столовой колледжа с адаптивным дизайном и функциональной админ-панелью. Приложение позволяет просматривать меню блюд, управлять ассортиментом через админ-панель, а также обеспечивает удобный интерфейс для пользователей с поддержкой темной/светлой темы.

## 📋 Описание проекта

ItHub-Menu - это веб-приложение для управления меню столовой колледжа, разработанное с использованием современных технологий. Приложение включает в себя:

- Адаптивный интерфейс, корректно отображающийся на устройствах любого размера
- Русскоязычный интерфейс
- Возможность просмотра меню блюд
- Админ-панель для управления блюдами
- Поддержка светлой и темной темы
- Тестовое блюдо "Борщ", автоматически добавляемое при запуске

## 📌 Стек технологий

- Java 21
- Spring Boot 3.5+
- Spring Data JPA
- Spring Security
- PostgreSQL (или H2 для тестирования)
- Liquibase
- Thymeleaf
- HTML5, CSS3, JavaScript (ES6+)
- Lombok
- Maven
- Docker & Docker Compose

## 🗄 Структура базы данных

Таблица dish:
- id (BIGSERIAL, PK) - уникальный идентификатор блюда
- name (varchar 255, not null) - название блюда
- price (int, not null) - цена блюда в рублях
- description (varchar 500, not null) - описание блюда
- image_url (varchar 500, not null) - URL изображения блюда

## 🎯 Функциональные возможности

### Для пользователей:
1. Просмотр меню блюд с изображениями, описанием и ценами
2. Адаптивный интерфейс для мобильных устройств (2 карточки в строке на мобильных экранах)
3. Переключение между светлой и темной темой
4. Кнопка оплаты через Сбербанк

### Для администраторов:
1. Аутентификация с защитой (логин: admin, пароль: admin)
2. Просмотр списка всех блюд
3. Добавление новых блюд
4. Редактирование существующих блюд
5. Удаление блюд
6. Адаптивная админ-панель для мобильных устройств

## 🛡 Безопасность

- Аутентификация с использованием Spring Security
- Один пользователь admin по умолчанию
- Логин: admin
- Пароль: admin
- Роль: ADMIN
- Доступ к /admin/** только для аутентифицированных администраторов
- Доступ к /menu и статическим ресурсам для всех пользователей

## 🚀 Запуск приложения

### Локальный запуск

1. Убедитесь, что у вас установлены Java 21 и Maven
2. Склонируйте репозиторий
3. Перейдите в директорию проекта
4. Выполните команду:

```bash
mvn spring-boot:run
```

5. Приложение будет доступно по адресу http://localhost:8082

### Запуск через Docker

1. Убедитесь, что у вас установлены Docker и Docker Compose
2. Склонируйте репозиторий
3. Перейдите в директорию проекта
4. Выполните команду:

```bash
docker-compose up --build
```

5. Приложение будет доступно по адресу http://localhost:8082

## 📱 Адаптивность

Приложение полностью адаптировано для мобильных устройств:
- На мобильных экранах (до 768px) отображаются 2 карточки блюд в строке
- На очень маленьких экранах (до 480px) отображается 1 карточка
- Уменьшенный масштаб элементов интерфейса для мобильных устройств
- Удобное меню навигации
- Адаптивная админ-панель
